Shift Duty Horror

Status: Planning Phase

Horror | Retro | 3D

An online co-op detective horror for 1 - 3 players. Work together to investigate your customers, as some are only pretending to be human.

One mistake, and you’ll be boarding up, setting traps, and hiding. Every run is randomly generated.

This project idea is heavily inspired by the existing "Shift at Midnight" game. None of the media in this page is mine.
